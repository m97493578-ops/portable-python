# Portable Python Custom Distribution

A pre-configured, lightweight, and completely portable Python distribution for Windows. Zero installation or administrator rights required.

## 📥 How to Download

1. Go to the **Releases** section on the right-hand sidebar.
2. Click on the latest version tag (e.g., `v3.14.5`).
3. Download the `.zip` archive file under the **Assets** dropdown.

## 🚀 How to Use

1. **Extract** the downloaded `.zip` file completely.
2. **Open** the extracted folder.
3. **Launch** your scripts directly using the enclosed `python.exe`.

## ⚙️ Features

* **Fully Isolated:** Will not conflict with existing system Python installations.
* **Portable:** Run it directly from a USB drive or network folder.
* **No Install:** Perfect for restricted corporate or school environments.
